• 正常 : normal
  • 近右 : right
  • 近左 : left
  • 近下 : down
  • 近上 : up
  • 随机 : random
  • 远右 : rightBig
  • 远左 : leftBig
  • 远下 : downBig
  • 远上 : upBig

调用方法：
$('选择器').typewriter({
text: '这是要输出的文本',
effect: 'left', /* 这是打印效果，正常 : normal
近右 : right
近左 : left
近下 : down
近上 : up
随机 : random
远右 : rightBig
远左 : leftBig
远下 : downBig
远上 : upBig */
excute: funtion() {console.log('这是打印期间执行的函数');}
complete: funtion() {console.log('这是打印结束后执行的函数');}
PS: 本插件依赖animate.css 和jq。
配置参数所有属性都是可选项。
此外，配置参数也可以是字符串：'stop'停止当前打印，'clear'清除所有打印文本。
